🪟 Windows NOC Tools

IP Público (WAN)
curl -4 ifconfig.me && curl -6 ifconfig.me
Configuração de IP e Gateway
ipconfig /all | findstr /C:"IPv4" /C:"Gateway" /C:"DNS"
Teste de Latência (Ping Contínuo)
ping 8.8.8.8 -t
Rastreio de Rota Rápido
tracert -d 8.8.8.8
Mapear Conexões Ativas (Processos)
netstat -ano | findstr ESTABLISHED
Tabela de Roteamento Local
route print
Forçar Rota Estática
route add 192.168.50.0 mask 255.255.255.0 192.168.1.254 -p
Teste de MTU (Caminho)
# Testa se o pacote 1500 passa sem fragmentar
ping 8.8.8.8 -f -l 1472
Tabela ARP (Conflitos de MAC/IP)
arp -a
Cache DNS Local
ipconfig /displaydns
Diagnóstico de DNS Externo
nslookup google.com 1.1.1.1
Captura de Pacotes (Nativa Windows)
pktmon start --etw                      # Inicia captura
pktmon stop                             # Para e gera log
Reset de Pilha TCP/IP e DNS
ipconfig /flushdns && ipconfig /release && ipconfig /renew && netsh winsock reset && netsh int ip reset
Limpeza de Cache ARP
arp -d *
Saúde do Sistema (SFC)
sfc /scannow
