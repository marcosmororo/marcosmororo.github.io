🪟 Windows Diagnostic Cheat Sheet

Script de Diagnóstico Completo (.BAT)
Gera relatório automático diag_%COMPUTERNAME%.txt
Baixar Script
Descobrir IP Público (v4 e v6)
curl -4 ifconfig.me && curl -6 ifconfig.me
Verificar Gateway (Roteador)
ipconfig | findstr "Gateway"
Teste de Latência Contínuo
ping 8.8.8.8 -t
Rastreio de Rota (Sem resolução de nome)
tracert -d 8.8.8.8
Teste de MTU (Fragmentação)
ping 8.8.8.8 -f -l 1472
Verificar Estatísticas da Placa de Rede
netstat -e
Listar Conexões Ativas (Processos)
netstat -abno
Reset Total de Rede (Flush/Reset)
ipconfig /flushdns && netsh winsock reset && netsh int ip reset
Renovar Endereço IP (DHCP)
ipconfig /release && ipconfig /renew
