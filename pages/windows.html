🪟 Diagnóstico Windows

Identificar IP Válido e WAN
Consulta rápida de IPs externos IPv4 e IPv6.
curl -4 ifconfig.me && curl -6 ifconfig.me
Traceroute Direto (Sem DNS)
Rastreio de rota ignorando nomes de host para maior velocidade.
tracert -d 8.8.8.8
Mapear Conexões com PID
Descobre qual Process ID está gerando conexões estabelecidas.
netstat -ano | findstr ESTABLISHED
Tabela ARP (Conflitos de IP)
arp -a
Persistência de Rota Estática
route add 192.168.10.0 mask 255.255.255.0 192.168.1.254 -p
Cache DNS Local
ipconfig /displaydns
Captura Nativa (Pktmon)
Inicia monitoramento de pacotes sem instalar Wireshark.
pktmon start --etw
Diagnóstico DNS (Servidor Específico)
nslookup google.com 8.8.8.8
