Mapear Conexões Ativas com PID (Process ID)
Utilitário para descobrir qual programa está "sequestrando" a banda ou abrindo conexões suspeitas.
netstat -ano | findstr ESTABLISHED
Análise de Tabela ARP (Detecção de IP Duplicado)
Verifica se o MAC Address do Gateway é o correto. Útil para identificar conflitos de IP na rede local.
arp -a
Persistência de Rota Estática
Forçar o tráfego de um destino específico para um Gateway alternativo.
route add 192.168.10.0 mask 255.255.255.0 192.168.1.254 -p
Dump de Configuração de Interface (Netsh)
Exporta todas as configurações de IP, DNS e rotas em formato de texto técnico.
netsh interface ip show config
Verificar Cache DNS Local
Veja o que o Windows já resolveu de DNS antes de consultar o servidor externo.
ipconfig /displaydns
Captura de Pacotes Nativa (Pktmon)
O Wireshark do próprio Windows (sem instalar nada). Captura tráfego em tempo real.
pktmon start --etw
Diagnóstico de DNS Específico
Forçar a consulta DNS usando um servidor específico (Ex: Google) para testar se o DNS do cliente está filtrando algo.
nslookup google.com 8.8.8.8
Relatório de Bateria/Energia (Útil para Laptops/ONTs via USB)
Gera um relatório HTML sobre a saúde da bateria (importante se o cliente reclama de queda ao flutuar energia).
powercfg /batteryreport
