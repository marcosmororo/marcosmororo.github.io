Ferramenta de Diagnóstico v2.1 (Windows)

Executa Ping, Tracert, MTU e coleta IPs Públicos automaticamente.

⬇️ Baixar .BAT
📖 Como utilizar
  1. Baixe o arquivo .bat acima.
  2. Clique com o botão direito e selecione "Executar como Administrador".
  3. Aguarde o término (a janela fechará ou pedirá para pressionar uma tecla).
  4. Envie o arquivo diag_NOME-DO-PC.txt gerado na mesma pasta.
🖥️ Visualizar Código (Batch)
@echo off
setlocal enabledelayedexpansion
title Diagnostico NOC Pro
set "FILE=diag_%COMPUTERNAME%.txt"
echo [1/4] Identificando IPs...
curl -s -4 https://ifconfig.me > temp.txt
set /p IP= %FILE%
echo [2/4] Testando Gateway...
ping 8.8.8.8 -n 10 >> %FILE%
echo [3/4] Rastreando Rota...
tracert -d -h 15 8.8.8.8 >> %FILE%
echo [4/4] Teste MTU...
ping 8.8.8.8 -f -l 1472 -n 1 >> %FILE%
pause
