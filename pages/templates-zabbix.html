1. Core (Backbone)

  • Juniper MX Router / QFX Switch / ACX
    • Informações: Hardware model name, Hardware serial number
    • Performance: Utilização de CPU, Utilização de Memória
    • Assinantes: Número total de assinantes PPPoE
    • Interfaces: Bits (In/Out), Estado (Admin/Oper), Erros, Descartes, Tipo, Velocidade
    • Óptico: Lanes (Bias, Tx/Rx, Temperatura, Voltagem)
    • Hardware: Status da FAN, Status da fonte de alimentação, Temperatura
    • Dependências: Module ICMP Ping Checker, Modulo SNMP Checker

2. Distribution (Agregação)

  • Huawei VRP Switch
    • Informações: Hardware model name, SO (Versão firmware/patch VRP)
    • Energia: Potência usada, Estado Fonte (PW1/PW2), Tipo Fonte, Presença Fonte
    • Interfaces: Bits (In/Out), Estado (Admin/Oper), Erros, Descartes, Tipo, Velocidade
    • Óptico: Single/Multi Lanes (Vendor, Pn, Tx/Rx, Temperatura)
    • Sistema: Uptime do Sistema, Status FAN
  • Raisecom Switch
    • Informações: Versão do hardware, Versão do software
    • Performance: Memória (Total, Disponível, %), Utilização da CPU
    • Hardware: Voltagem, Status/Speed da FAN, Temperatura do equipamento
    • Interfaces: Erros CRC
    • Óptico: Lanes (Bias, Tx/Rx, Temperatura)
    • Dependências: FB - Interfaces, ICMP Ping, SNMP Checker, Generic by SNMP
  • ZTE Switch
    • Óptico: FB - Interface otica ZTE (Status, Temperatura, TX/RX)
    • Dependências: Network Generic Device, Template SNMP Location, Module Interface Discovery v2.0

3. Acesso

  • DmOS DM4370 4GT+4GX+4XS
    • Informações: Serial number
    • Dependências: DmOS (Contact, Desc, Name, Uptime), ICMP Ping, Interface Discovery v1.0, DmOS (CPU, FAN, Memory, Temperatura)
  • DATACOM - v2
    • Informações: Model name, Modelo/Nome do Produto, Modo Operação, Serial, Número de Portas, Versão Bootloader/Firmware
    • Performance: Uso de CPU, Uso de Memoria
    • Hardware: Status Energia/Power Status, Temperatura, Alarm Status, FAN Status
    • Interfaces: Admin status, In/Out, Erros, Descartes, Status interface
    • Óptico: Lanes (Bias, Tx/Rx, Temperatura, Voltagem)
    • Dependências: Module Hardware and software information
  • Mikrotik - RSW
    • Dependências: ICMP Ping, Module Checker (Ver/Firm), Health Check Voltage, Interface Discovery, SNMP Checker

4. OLTs

  • ZTE C600
    • System Information (Uptime, Firmware)
    • LLD: Unidades de Processamento (Cards) / Temperaturas / CPU / Memória
    • LLD: Hardware Health (Fontes / Fans)
    • LLD: Interfaces de Rede (Trafego / Status / Erros)
    • LLD: Monitoramento PON (Status Portas / Métricas ONUs)
    • Dependências: FB - Interfaces OLT ZTE
  • Fiberhome
    • LLD: Interfaces de Rede (Módulo Porta PON Tráfego / XFP Info Card)
    • System (Uptime, Global CPU)
    • LLD: Slots & Cards (Status / Performance CPU-Mem %)
    • LLD: PON Interfaces (Diagnóstico Óptico TX/Temp/Volt)
    • Censo: Total de ONUs
    • Tráfego: Trigger 2Gbps Download/Upload
  • Nokia
    • Management: Fan Mode/Speed, Temp NT-B
    • LLD: Placas de Serviço (Status Hardware, CPU/Mem per Slot)
    • LLD: Interfaces PON (Censo Provisionado/Online/Offline)
    • Trigger: 30% Mass Failure
    • LLD: Portas PON (Oper/Admin Status, Alias)
  • V-Sol
    • Hardware Monitoring: CPU, RAM, Temp Chassis
    • LLD: Interface Discovery (Uplinks) / PON Discovery (Traffic, ONUs Online)

5. Retificadores

  • Delta Retificador
    • Bateria: Tensão, Corrente, Temperatura, Estado de Carga
    • Carga: Potência, Corrente de Carga
    • Retificadores: Nº de URs falhas/operacionais, Corrente UR
    • Tempos: Autonomia, Duração restante
    • Dependências: Module ICMP Ping Checker
  • JFA Retificador 40A-48V
    • Autonomia, Corrente Saída, Nível de carga
    • Elétrico: Temperatura Interna, Tensão Rede VCA/Saída
  • XPS Retificador (SRWi / SRX)
    • Bateria: Status (Carga/Descarga/Desc), Amperagem, Autonomia
    • Alarmes: Falha AC, Falha UR, Fusível Aberto
    • Dependências: ICMP (Jitter/Loss), Module Autonomy time

6. Módulos Genéricos

  • Module ICMP Ping Checker: Jitter, Latência, Perda, Status
  • Module SNMP Checker: Contato, Disponibilidade, Uptime
  • FB - Interfaces: Bits, Status, Erros, Descartes
