Central de Automação NOC

Baixe a ferramenta de limpeza de ONUs para a OLT Nokia.

⚙️ Preparação necessária:

1. Certifique-se de ter o Python instalado.
2. Antes de rodar o script pela primeira vez, instale a biblioteca necessária via terminal:
pip install netmiko

RECOMENDADO
Script Python limpeza_olt.py (V1.0)
Fluxo de Trabalho: Execute o script, insira as credenciais da OLT e aguarde.
O arquivo remover_onus.txt será gerado na mesma pasta com os comandos de remoção.
