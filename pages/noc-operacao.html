Validação de Latência PtP

Ferramenta de teste de login SSH e monitoramento de performance ponto-a-ponto.

⚠️ Requisitos de Sistema:

1. SSH habilitado no equipamento (Porta 22).
2. Python 3.x instalado localmente.
3. Dependência SSH: pip install paramiko netmiko

ESTÁVEL
Script de Teste teste_latencia_ptp.py
Login Automático Validação via Porta 22
O que este script executa:
  • Autenticação segura no equipamento via protocolo SSH.
  • Disparo de pacotes ICMP (Ping) para o destino PtP definido.
  • Captura de Latência de Referência (Benchmark).
  • Comparação em tempo real: Latência Atual vs. Referência.
  • Relatório de desvio (Jitter e variação de ms).
Dica: Utilize este script para identificar degradações de rota ou problemas físicos no enlace
comparando o estado atual com os parâmetros ideais salvos.
